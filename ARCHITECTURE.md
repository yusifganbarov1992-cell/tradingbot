## Component Details

### 1. User Interface Layer

#### Telegram Bot (`trading_bot.py`)
- Primary user interaction point
- Command handling for all trading operations
- Real-time notifications
- Operator chat for alerts

#### Streamlit Dashboard (`dashboard/app.py`)
- Visual analytics and reporting
- Portfolio performance tracking
- Historical trade visualization
- Real-time market data display

### 2. Core Trading Engine

#### TradingAgent
- Central orchestrator for all trading operations
- Coordinates between analysis, safety, and execution
- Manages active positions
- Handles market scanning

#### SafetyManager (8 Levels)
1. Position Size Limits
2. Portfolio Correlation
3. Volatility Check
4. Liquidity Verification
5. Time-Based Rules
6. Drawdown Protection
7. Risk Score Assessment
8. Emergency Override

#### AutonomousTrader
- Automatic trade execution
- Confidence-based decisions
- Hourly trade limits
- Whitelist/Blacklist management

### 3. AI/ML Layer

#### OpenAI Integration
- GPT-4o-mini for market analysis
- Natural language processing
- Sentiment extraction
- Risk assessment

#### Technical Analysis
- RSI, MACD, Moving Averages
- Volume analysis
- Support/Resistance levels
- Pattern recognition

### 4. Data Layer

#### Supabase (Cloud)
- Primary data store
- Real-time sync
- Trade history
- Settings persistence

#### Local SQLite
- Offline capability
- Fast caching
- Session management

### 5. External Services

#### Binance API (ccxt)
- Exchange connectivity
- Order execution
- Balance queries
- Market data
